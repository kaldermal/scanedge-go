# ScanEdge Go - Генерирует IP адреса, и проверяет его доступность, а также доступность введённых портов

## Команды:
`.unl` - Убрать лимит в количестве IP, вместо 1000
это в интерактиве

АРГУМЕНТЫ:
./scanedge {скоко айпи} {порты} {таймауты или я хз как называется}
ну пример
./scanedge 1000 80,443,22 0.5
